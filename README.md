# основной Класс
```c++
class MathOperation {
public:
    virtual ~MathOperation() = default;
    virtual double calculate(double a, double b = 0) const = 0;
    virtual string getName() const = 0;
    virtual bool isBinary() const = 0;
};
```
Пример
